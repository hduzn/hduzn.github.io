Linux vim/vi 命令简单使用

文章目录
  1. 1.两种模式切换
  2. 2.保存文件
  3. 3.命令 功能
  4. 4.小结

vim平常如果很少使用，偶尔用用的，我觉得掌握这点就够用了。

vim有两种模式，一种是命令模式，另一种是编辑模式。

在命令模式下，只能输入命令操作，并不能输入内容；
在编辑模式下，则能正常编辑内容。

1.两种模式切换

例如 运行 vim 1.txt 后，即进入了命令模式，此刻不能输入内容。

  • 当按下 i 键后，即进入编辑模式，可输入内容。
  • 再按 Esc 键后，退出编辑模式，回到命令模式。

当进入编辑模式后，页面最下方会有显示 – INSERT – 的提示，表示当前是编辑模式。

2.保存文件

需要在命令模式下输入 :wq 后回车即保存退出。

命令 功能
:w 保存
:wq 保存并退出
:q 不保存退出

3.命令 功能

命令 功能
o 插入一行
dd 删除一行
dw 删除选中字母开始的第一个字母

4.小结

简单的使用，会这么4条就行了。

  • 1.用vim 打开文件后，是命令模式；
  • 2.按 o 键增加新的一行，会自动进入编辑模式；
  • 3.编辑完内容后，按 Esc 键回到命令模式；
  • 4.最后输入命令 :wq 回车，保存退出或者 :q不保存退出。
