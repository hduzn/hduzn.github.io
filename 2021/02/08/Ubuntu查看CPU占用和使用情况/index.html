Ubuntu 查看CPU占用和使用情况
HDUZN
  2021-02-08 21:00:00 2021-02-08 21:00  
  • Linux 
 
  • Linux 
  • | Ubuntu 
 

使用自带的 top命令和 htop命令就可以查看，个人推荐第2种。

方法一：用自带的 top命令查看

  • 第1行：当前系统时间，系统已运行的时间，几个在线用户(user)；
  • 第2行 Tasks：总进程数(total)，正在运行的进程数(running)等；
  • 第3行 %Cpu：用户进程 CPU占用率(us)，系统进程 CPU占用率(sy)，CPU空闲率(id)；
  • 第4行 KiB Mem：内存总量(total) ，内存空闲(free)。单位都是KB。上图中内存约1.8 G。
  • 第5行：略
  • 第6行 PID：显示具体进程情况。%CPU是CPU情况，%MEM是内存使用情况。

方法二：使用 htop查看（推荐使用）

1.安装 htop

2.使用htop命令可以更直观的查看CPU使用信息

- 上方左侧：CPU、Mem(内存) 看得更清楚明白。内存总共 1.79G，已使用 159M。 - 上方右侧：全部进程(Task)数为：33。 - 下方一样的是全部进程的情况。

如果想要结束进程，按上面的提示按F9(Kill) 就可以结束进程，需要再按回车键(Enter)确认。
