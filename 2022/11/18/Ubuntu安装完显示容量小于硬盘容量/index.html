Ubuntu安装完显示容量小于硬盘容量
HDUZN
  2022-11-18 21:59:56 2022-11-18 21:59  
  • Linux 
 
  • Linux 
  • | Ubuntu 
 

Ubuntu Server 20.04安装完后（安装笔记：U盘安装Ubuntu Server），发现安装的这块硬盘有256G，但明显只看到100G。因为安装的时候都直接默认了。

解决方法：

1.用 lvdisplay 命令查看LVM逻辑卷信息

可以看到结果中（LV Path记一下，后面会用到）：

LV Path /dev/ubuntu-vg/ubuntu-lv
LV Size 100.00 GiB

2.用 fdisk -l命令查看所有分区表

结果如下，可以看到硬盘实际大小是238.49G（256G硬盘）是SAMSUNG三星的。明显是大于100G的。

3.逻辑分区扩容

结果如下：

看到 /dev/mapper/ubuntu–vg-ubuntu–lv 232G这里已经有了。
