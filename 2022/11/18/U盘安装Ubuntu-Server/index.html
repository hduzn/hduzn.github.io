U盘安装Ubuntu Server
HDUZN
  2022-11-18 21:11:07 2022-11-18 21:11  
  • Linux 
 
  • Linux 
  • | Ubuntu 
 

想在办公室搞一台Ubuntu Server，然后装上docker后，就可以搭一些小服务了。正好有同事换下来的电脑，配置还可以，8G内存+256G固态的，一个挺小的机箱。

查了一下，安装也挺方便的，下载系统包，然后用U盘制作个启动盘，再安装就行了。

一、下载Ubuntu Server的ISO镜像包

官网下载：https://ubuntu.com/download/server
官网目前最新版本：22.04.1

之前用的是18.04和20.04的，所以还是想下载用20.04版本的。于是，可以去清华大学的开源软件镜像站下载：https://mirrors.tuna.tsinghua.edu.cn/ubuntu-releases/20.04/

找到server版本下载：ubuntu-20.04.5-live-server-amd64.iso
desktop是桌面版本的，自己用也可以用桌面版的，反正我就当当服务器，用不着桌面版的，直接就server版。

二、制作U盘启动盘

1.下载制作启动盘的工具软件：Rufus

这个小软件特别小，且好用，无需安装，直接运行就可以用。Win7以上操作系统都可以用。

官网下载地址：http://rufus.ie/zh/
目前最新的是：3.20版本（1.3MB）

2.制作启动盘

插上U盘，运行Rufus后，

Device（设备）：选择U盘
Boot selection（引导类型选择）：点击SELECT（选择）按钮，添加下载好 ubuntu-20.04.5-live-server-amd64.iso

其它都默认就行，直接点击START（开始）。
期间出来一个提示，要下载2个文件啥的直接下载就行。

然后等待完成就好了。

三、安装Ubuntu Server

1.安装ubuntu 20.04.5 LTS

插上U盘，跟安装Windows一样，选择从U盘启动就行。

安装过程都直接默认就行。其中，

  • 网络是没设置的，选的是Continue without network，安装完再设置。
  • 硬盘设置，看到disk的地方，如果有2块硬盘的，选一块硬盘。

除了选了一块硬盘，别的都默认了。其它的搞不懂可以安装完再整。

最后面有个登录账号信息需要填写一下：

等待安装完成后，选一下【Reboot Now】（重启），选完就拔掉U盘启动。

2.设置网络（静态IP）

安装完成重启后，可以先查看发行版本信息：

插上网线后，用vi打开配置文件/etc/netplan/00-installer-config.yaml（不同版本可能文件名有点不一样）

配置文件内容如下（vi命令不会的看这篇笔记：Linux vim/vi 命令简单使用）：

dhcp4: no 关闭ipv4自动获取。
addresses: [192.168.1.175/24] 在指定IP地址时，ip地址之后的子网掩码是必须的，在使用ipv6时也一样必须带有子网掩码，以上ipv4地址的掩码是24。
gateway4：字段是网关字段，即路由器的地址。
nameservers字段：设置域名服务器IP地址。

好了之后，保存、退出。

启动服务：

网络正常了，那就可以在别的电脑（同局域网内）SSH连接啦，就不需要显示器了，开着主机就行了。
