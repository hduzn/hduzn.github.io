Edge浏览器无法同步 解决方法
HDUZN
  2022-06-19 13:25:33 2022-06-19 13:25  
  • Windows 
 
  • Windows 
  • | Software 
 

办公室电脑上用的Edge浏览器，然后家里电脑上打开Edge浏览器的时候，发现总是没办法自动同步收藏夹、历史记录这些。（用的是同一个微软用户的邮箱账号登录的）

用别的浏览器，明显是可以都同步的。

当然，在浏览器【设置】-【同步】设置中，这些同步的选项都是启用的。

解决方法：

1.在Edge浏览地址栏中输入以下地址，回车。

如下图：

Server URL中如果无数值，此时设备不能同步。

按照网上的说法，可以修改Windows的hosts文件，添加以下内容：

再打开edge://sync-internals/，如果可看到Server URL中有数值，说明可以正常同步。

但，我的问题是明明有Server URL，但是就是没同步过来。

2.关闭同步，重新开启同步。

像我这样的问题的话，可以直接在两边电脑上都先在上面的页面中关闭同步【Stop Sync（Keep Data）】，或者去【设置】->【同步】选项中关闭也是一样的。

然后依次重新开启同步，重新开启同步后，默认没有选上“收藏夹”、“历史记录”这些选项的，手动选上。

然后终于同步过来了。

感觉Edge的同步就跟用微软的产品差不多，时不时能遇到抽风的时候。要不是我这可以远程连接，操作另一台电脑，这要是没同步过来，不都歇菜了么。

目前，能正常同步了。

最后，特别要注意的是，千万千万不要去点【重置同步】，这一重置，直接就把云数据清空了！
