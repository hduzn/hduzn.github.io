我有一台腾讯云的CentOS 7.6，不行，安装依赖库的时候直接对应的腾讯云的mirrors都空了，没有。

于是又试了下，另一台CentOS 8.2版本的，正常可以安装。

Git官网对于Linux系统安装Git的说明：
https://git-scm.com/download/linux

别的系统都很简单，直接一条命令就行。
但就是这个CentOS这几个，需要自己下载源码、编译才能安装。

用yum install git命令安装的是老版本，反正都不行的。

CentOS安装git方法

1.官网下载源码

上面提供的官网（https://git-scm.com/download/linux ）最后CentOS部分，提供了源码下载的链接：
https://mirrors.edge.kernel.org/pub/software/scm/git/

然后直接找一下对应的新版本，我这里下载的新版本是：git-2.36.0.tar.gz
