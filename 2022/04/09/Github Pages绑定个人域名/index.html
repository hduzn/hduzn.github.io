Github Pages绑定个人域名
HDUZN

  • Hexo 
 
  • Github 
  • | Hexo 
 

用Github Pages + Hexo搭建个人博客后，域名都是[github用户名].github.io，如果想换成自己的域名，比如我的github上的站点是：hduzn.github.io，要绑定的域名为：www.i007it.com

1.在Github上进行设置

  • 打开Github上[github用户名].github.io的配置
  • 找到【Settings】中，页面拉到最下面【Github Pages】选项，
  • 在Custom domain项中输入域名：i007it.com后，Save保存一下。

2.添加CNAME文件

3.修改DNS中对应的IP地址

在域名解析服务商中，将域名的A记录对应的ip地址设置为github.io对应的ip。

1).先通过ping github.io 命令，得到对应的ip地址为：185.199.110.153 （不固定，需要自己ping一下）

2).当域名解析(DNS)服务商中的A记录对应ip地址设置为上面ping出来的ip地址。

这样就可以了。
